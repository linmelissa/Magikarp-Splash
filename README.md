# Magikarp-Splash

## Description
A 2D Platformer game that is Pokemon themed created in Python with Pygame

![](/assets/gameplay.png)

## Instructions
- SPACE to jump
- Eat rare candies to level up
- Avoid Remoraids
- Reach level 20 to evolve into Gyarados
